# Controller Plan Index

This folder stores the deferred Bluetooth gamepad plan for QuestRetroDepth.

Files:
- `standard_gamepad_support_plan.md`: end-to-end feature spec and exact code changes.
- `controller_mappings.md`: fixed gameplay mappings for SNES and Genesis.
- `rumble_and_safety.md`: controller detection, safety gate, rumble behavior, and validation checklist.

Status:
- Planning only.
- No implementation has been applied to the app yet.
